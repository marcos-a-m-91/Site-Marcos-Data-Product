06 · Animations

Motion
Grammar

Three scroll-parallax planes, mouse-driven tilt, and GSAP scroll-triggered reveals define the motion grammar.

Scroll Parallax

Three depth layers move at 25×, 15×, and 8× scroll multipliers for realistic spatial separation.

group[0].y = scrollY * 25
group[1].y = scrollY * 15
group[2].y = scrollY * 8

Mouse Tilt

Cursor position rotates parallax groups with diminishing influence per layer — foreground reacts most.

group[0].rot = mouse * 0.12
group[1].rot = mouse * 0.06
group[2].rot = mouse * 0.03

Scroll Reveal

GSAP ScrollTrigger fades and translates elements into view as they enter the 80% viewport threshold.

ease: "power3.out"
duration: 0.9s
delay: 0.25s stagger

Animation Reference

GSAP ScrollTrigger

ScrollTrigger.create({
  trigger: el,
  start: "top 88%",
  onEnter: () => gsap.to(el, {
    opacity: 1,
    y: 0,
    duration: 0.85,
    ease: "power3.out"
  }),
  onLeaveBack: () => gsap.to(el, {
    opacity: 0,
    y: 32,
    duration: 0.5
  })
});
